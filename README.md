# SSKJ-Android
上善科技智能污水处理系统安卓客户端

## 前言
此项目试图开发出一个智能污水处理设备监控及配套产品销售的应用，现阶段主要以 **网页（JAVA WEB）**、**Android**客户端的开发为主

## 功能目标
- [ ] 首页信息展示
- [ ] 水质监测
- [ ] 商城系统

## 现状
- v0.0.1 - 项目初始化
- v0.0.2 - 基本框架构建
- v0.0.3 - 基本界面绘制

## 运行环境
### Android 5.1 - Android P

## 开发环境
> Android Studio 3.1.2 <br>
> TargetSDK: 27 <br>
> MinSDK: 22 <br>
> Kotlin 1.2.41 <br>
> Anko 0.10.5
